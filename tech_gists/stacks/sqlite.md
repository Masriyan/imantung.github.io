# SQLite

Ref:
- <https://www.sqlite.org/cli.html>

Installation
```bash
# mac
brew install sqlite

# linux
sudo apt install sqlite3
```


Cheatsheet Interactive Command/Dot Command

| Command | Description |
| ------------- |-------------|
| `.help` | Show Help |
| `.quit` | Quit from Interactive Command |
| `.tables` | Show tables |
| `.mode list` | Result as list |
