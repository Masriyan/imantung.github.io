Line Indicator:
- `|` Verbatim Text (support HTML)
- `'` Verbatim Text + single trailing white space
- `<` beginning of regular html tag
- `-` Control Code
- `=` Output
- `==` Output without HTML escaping
- `/` Comment
- `/!` Html comment
